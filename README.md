# Duckov Drinks 鸭子饮品包
本模组是《逃离鸭科夫》的一款专精于饮品和饮品制作的模组。

## 模组内容
本模组目前版本添加了四种饮料：酸梅汤、纯牛奶、营养快线和冰红茶。
部分饮料可以通过现有的材料进行制作，也可以通过搜刮等方式获得。
内容持续拓展中，敬请期待！
